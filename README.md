# CodaBlog
CodaBlog is a full-stack blog MERN web application built in JavaScript. 
It allows for CRUD functionality. A user can create, read, update, or delete a blog post. 

## Technologies
- MongoDB
- Express
- React.js
- Node.js
- HTML/CSS
- JavaScript
